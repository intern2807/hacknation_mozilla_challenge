How It Works

The bookmarklet uses the Harbor JavaScript API. Here's the core logic:

// Check if Harbor is available if (!window.agent || !window.ai) { alert('Harbor extension not found!'); return; } // Request permissions await window.agent.requestPermissions({ scopes: ['model:prompt'], reason: 'Summarize this page' }); // Get page content (using DOM, not activeTab) const text = document.body.innerText; // Create AI session and summarize const session = await window.ai.createTextSession({ systemPrompt: 'Summarize clearly and concisely in 2-3 paragraphs.' }); const summary = await session.prompt(text); // Display in overlay showOverlay(summary);

Note: Since the bookmarklet runs on the page, it reads document.body.innerText directly instead of using activeTab.read.
